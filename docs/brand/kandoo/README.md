# Kandoo mascot assets

The `expressions/` directory contains 1024 px transparent PNG masters. Runtime-sized copies live under `src/assets/kandoo/expressions/`.

Expression semantics:

- `calm`: canonical brand mark and ordinary information
- `success`: completed actions
- `loading`: loading, saving, connecting, and syncing
- `error`: failed actions and sync conflicts
- `warning`: invalid input and non-fatal attention states
- `online`: connected and cloud-synced
- `offline`: local-only, guest, or disconnected

The `icon/` directory contains the four deterministic SVG design layers, the Icon Composer bundle, flattened appearance previews, and the legacy ICNS source. Clear and tinted renditions are previews of Icon Composer's mono appearance; the operating system supplies the final glass, tint, lighting, and refraction.
